E-commerce Site

Click to view website

For a web development assignment in second year I produced a simple e-commerce site with PayPal implmentation that allows users to purchase merchandise online. It also had to adhere to WCAG 2.0 AA Accessibility standards. The functionality of the website is as follows:

  • Restict areas of the website to specific users
  • Add/edit products
  • Upload images for products
  • Ability to have up to 2 images per product
  • "Purchase" products (PayPal is setup in sandbox mode)
  • Contact site owner via contact page
  • Change account password
  • Google Maps integration

The project was built using ASP.NET web forms, therefore HTML, CSS and C# were the languages used. SQL was used to handle database requests.
