Spirit Data Capture Ltd. - Vehicle Module (App)

Click to view website Click to view accompanying web portal

During the final term of second year at University I led a small team of 3 (including myself) to produce both a Web Portal and Android Application. This project lasted six weeks where I acted as both team leader and lead developer. This meant that 90% of the code within these two projects was created by myself, with some tweaks by the other team members. The portal has the following functionality:

  • Restict areas of the website to specific users
  • Add/remove vehicles
  • Add/edit vehicle types
  • Add/edit checks associated to vehicle types
  • View/search Completed vehicle checks
  • Add/edit users

The project was built using ASP.NET web forms, therefore HTML, CSS and C# were the languages used. SQL was used to handle database requests.
